# 📚 BookHaven

Un gestionale completo per biblioteche, pensato sia per gli utenti (studenti/ragazzi) che per lo staff amministrativo. 
Gli utenti possono richiedere una propira tessera e creare il proprio account facendosi registrare dallo staff,visualizzare articoli di blog o eventi,
prenotare libri e visualizzare il proprio storico.
I membri dello staff possono gestire il catalogo,registrare nuovi utenti, le prenotazioni dei libri, creare nnuovi articoli di blog e monitorare l'attività della biblioteca.

## 🚀 Funzionalità principali

### 👤 Utenti (Studenti)
- Login
- Ricerca e visualizzazione dei libri disponibili
- Prenotazione dei libri
- Visualizzazione dello storico delle prenotazioni

### 🛠️ Staff e Amministratori
- Login riservato
- Registrazione utenti
- Gestione del catalogo (aggiunta, modifica, rimozione libri)
- Monitoraggio delle prenotazioni
- Gestione degli utenti registrati
- Visualizzazione delle statistiche d'uso

## 🏗️ Tecnologie utilizzate

- **Backend**: Java Spring Boot
- **Frontend**: React / Redux / Bootstrap / CSS
- **Database**: PostgreSQL
- **Autenticazione**: JWT 
- **Altro**:API REST
